📚 在线学习系统

基于 Spring Boot 的在线学习平台，支持课程讲座管理、投票系统和用户权限管理。

Spring Boot 3.x Spring Security Spring Data JPA H2/MySQL Gradle

✨ 功能特性

  • 📚 讲座管理 - 创建、编辑、删除讲座，支持附件上传
  • 🗳️ 投票系统 - 创建投票、用户投票、实时结果
  • 👥 用户管理 - 注册登录、角色权限控制
  • 💬 评论功能 - 讲座和投票的评论系统

🚀 快速开始

git clone https://github.com/你的用户名/仓库名.git
cd 仓库名
./gradlew bootRun

📁 项目结构

src/main/java/hkmu/wadd/
├── controller/     # 控制器层
├── dao/           # 数据访问层
├── service/       # 服务层
├── model/         # 实体类
├── config/        # 配置类
└── exception/     # 自定义异常

🔧 部署方式

本项目是 Spring Boot 后端应用，无法在 GitHub Pages 上运行。请使用以下平台部署：

  • Railway - railway.app（推荐，简单免费）
  • Render - render.com
  • Fly.io - fly.io
